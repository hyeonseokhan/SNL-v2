# 효율 계산 상수 테이블

> LOPEC (lopec.kr) 시뮬레이터 역추적 기반
> 최종 수정: 2026-04-03

## 개요

LOPEC 효율 계산은 표시용 문자열이 아닌 **내부 상수값**으로 동작합니다.
각 옵션은 구조화된 속성(`finalDmg`, `critRate`, `addDmg` 등)을 가지며,
이 값들을 조합하여 최종 효율(로펙 점수)을 산출합니다.

---

## 1. 핵심 스탯 변환 공식

```
치명 → 치적:     floor(critStat / 0.2794) / 100    (27.94 치명 = 1% 치적)
특화 → 딜증:     특화 / 20.791 / 100 + 1
신속 → 쿨감:     0.0214739 × 신속 / 100
치명타 기본 피해: 200%
```

## 2. 악세서리 연마 옵션

각 옵션은 `유물 하/중/상` + `고대 상` 4단계 값을 가집니다.

### 효과 속성과 값

| 옵션 | 속성 | 유물 하 | 유물 중 | 유물 상 | 고대 상 |
|------|------|---------|---------|---------|---------|
| 추가 피해 | `addDmg` | 2.5% | 3.0% | 3.5% | 4.0% |
| 적에게 주는 피해 | `finalDmg` | 1.015 | 1.02 | 1.025 | 1.03 |
| 적주피 + 무력화 | `finalDmg` | 1.015 | 1.02 | 1.025 | 1.03 |
| 쿨 +2% · 적주피 | `finalDmg`, `cdr` | 1.04 | 1.045 | 1.05 | 1.055 |
| 치적 + 치주피 | `critRate`, `critFinalDmg` | 2.6%/1.015 | 3.4%/1.015 | 4.2%/1.015 | 5.0%/1.015 |
| 치피 + 치주피 | `critDmg`, `critFinalDmg` | 5.2%/1.015 | 6.8%/1.015 | 8.4%/1.015 | 10.0%/1.015 |
| 공이속 | `atkSpeed`, `moveSpeed` | 3% | 4% | 5% | 6% |
| 무공 (고정) | `weaponAtkPlus` | 8100 | 9200 | 10300 | 11400 |
| 무공 (스택 6중첩) | `weaponAtkPlus` | 6000 | 6960 | 7920 | 8880 |
| 무공 (30중첩) | `weaponAtkPlus` | 7020 | 8005 | 8990 | 9975 |
| 추피 + 악마 | `addDmg`, `devilDmg` | 2.0%/1.025 | 2.5%/1.025 | 3.0%/1.025 | 3.5%/1.025 |
| 백어택 피해 | `backDmg` | 1.02 | 1.025 | 1.03 | 1.035 |
| 헤드어택 피해 | `headDmg` | 1.02 | 1.025 | 1.03 | 1.035 |

## 5. 장비 품질/재련

### 무기 품질 배율 (quality → multiplier)

```
 0: 1.1000    50: 1.1500
10: 1.1020    60: 1.1720
20: 1.1080    70: 1.1980
30: 1.1180    80: 1.2280
40: 1.1320    90: 1.2620
              100: 1.3000
```

### 레벨 배율

```
Lv 55-59: 1.0895
Lv 60-64: 1.1856
Lv 65-69: 1.2397
Lv 70:    1.2945
```

### 공격력 공식

```
최종공격력 = floor(sqrt(powerIndex × weaponAtk / 6) × atkBonus + flatAtkSum) × atkPercentMultiplier)
```

## 6. 아크패시브 포인트

### 전투력 배율 공식

```
진화:     75 × max(points - 40, 0) / 10000 + 1
깨달음:   min(70 × max(points, 0), 7000) / 10000 + 1
도약:     20 × max(points, 0) / 10000 + 1
```

### 도약 보석 효과 배율

```
points >= 70: x1.15
points >= 68: x1.14
points >= 66: x1.13
points >= 64: x1.12
points >= 62: x1.11
points >= 60: x1.10
points >= 50: x1.05
points >= 40: x1.03
otherwise:    x1.00
```

### 내실 포인트 값

```
없음: 0, 일반1: 3, 일반2: 6, 일반3: 9
상급: 5, 상급+일반1: 8, 상급+일반2: 11, 상급+일반3: 14
```
