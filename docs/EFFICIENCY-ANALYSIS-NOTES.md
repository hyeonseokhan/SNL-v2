## LOPEC 각인 효율 계산 방식 분석

### 발견 사항

LOPEC 각인 효율은 **두 가지 다른 계산 방식**을 사용합니다:

1. **전투력 기여도 (ABILITY_ATTACK 테이블)** — 로펙 점수 산출용
   - `key = 20 × stoneLevel + level + gradeOffset` 로 조회
   - `point / 10000 + 1` = 개별 승수
   - 곱연산으로 총합 계산

2. **실제 딜증 수치 (효율표 표시용)** — 풀 딜 시뮬레이션 기반
   - "각인 효율은 로펙 점수에 적용되는 값이 아닌, 실제 각인의 딜증 수치를 표시합니다"
   - "모든 각인은 팔찌와 마찬가지로 해당 직업, 세팅의 주력기를 기준으로 계산합니다"
   - 각인 포함/제외 시 총 딜 시뮬레이션 비교

### 차이 원인

| 각인 | 전투력 기여도 | 딜 시뮬레이션 | 차이 원인 |
|------|------------|-----------|---------|
| 돌격대장 | 22.96% | 22.95% | 거의 동일 (이속 고정) |
| 아드레날린 | 24.38% | 29.13% | 치적 20%의 딜 기여가 포함됨 |
| 타격의 대가 | 17.00% | 15.83% | 백헤드 불가 스킬 비율에 따라 다름 |
| 질량 증가 | 17.00% | 17.20% | 공속 감소의 쿨감 영향 포함 |

### 현재 구현 상태

- **전투력 기여도 방식**: ABILITY_ATTACK 테이블 기반 구현 완료 (개별 값 일치)
- **딜 시뮬레이션 방식**: 미구현 — 스킬별 모션 계수, 딜 비율 등 필요

### 딜 시뮬레이션 구현에 필요한 데이터

1. 스킬별 모션 계수 (로스트빌드 API)
2. 스킬별 딜 비율 (사용 빈도)
3. 스킬별 타입 (백어택/헤드어택/일반)
4. 스킬별 치적 보정 (급소 노출 등)
5. 각인 제거 시 연쇄 변화 계산 (돌격대장 ↔ 이속, 아드레날린 ↔ 치적+공격력)

## 질풍노도 (기상술사) 공식 정리

로스트빌드 역분석 기반 확인된 공식:

### 치적
- 질풍노도 기본: `10% + floor(0.3 × min(이속증가, 40) × 100) / 100`
- 이속증가 = 신속 + 직업 + 버프 (기본 14% 제외하지 않은 전체)
- min(53.83, 40) = 40 → `10 + 12 = 22%`
- 기민함/급소 노출은 LOPEC 효율표에 미포함 (스킬별 딜 계산에서만 사용)

### 치피
- 기민함 Lv3: `min(공속, 40) × 40 × level / 10000 × 100`
- min(43.83, 40) = 40 → `40 × 40 × 3 / 10000 × 100 = 48%`

### 음속 돌파 Lv2
```
i = min(공속, 40) × 0.1 + min(이속, 40) × 0.1
o = max(0, 공속 - 40) × 0.3 + max(0, 이속 - 40) × 0.3  
bonus = (공속 > 40 && 이속 > 40) ? 8 : 0
total = min(floor((i + bonus + o) × 100) / 100, 12 × level)
```
