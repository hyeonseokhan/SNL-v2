# 시뮬레이터 입력 필드 명세

> LOPEC 시뮬레이터 (lopec.kr) 기반 분석
> 최종 수정: 2026-04-03

## 개요

시뮬레이터의 목적: 현재 캐릭터의 세팅 값을 변경하여 효율(로펙 점수, 전투력 등)이 어떻게 변하는지 확인.
모든 기본값은 API에서 가져온 캐릭터 실제 데이터를 사용합니다.

---

## 1. 보석 (11슬롯)

각 슬롯당 3개 필드:

| 필드 | 입력 타입 | 값 |
|------|----------|-----|
| 보석 종류 | dropdown | 없음, 겁화(딜증+), 작열(쿨감+), 멸화(딜증), 홍염(쿨감) |
| 보석 레벨 | dropdown | 1~10 |
| 적용 스킬 | dropdown | 클래스별 스킬 목록 |

**일괄 변경:** 10겁작, 9겁작, 8겁작, 7겁작, 6겁작

---

## 2. 장비 (6부위)

각 부위(투구, 어깨, 상의, 하의, 장갑, 무기)당:

| 필드 | 입력 타입 | 값 |
|------|----------|-----|
| 티어 | dropdown | T4 유물, T4 고대, T4 전율 |
| 일반 재련 | dropdown | +0 ~ +25 |
| 품질 | number input | 0~100 |
| 상급재련 | dropdown | X24~X40 (조건부 활성) |

**일괄 변경:** 일반 재련 일괄, 상급재련 일괄

---

## 3. 아바타 (4부위)

| 필드 | 입력 타입 | 값 |
|------|----------|-----|
| 아바타 등급 | radio | 없음, 영웅, 전설 |

부위: 무기, 투구, 상의, 하의

---

## 4. 악세서리 (5종)

각 악세서리(목걸이, 귀걸이x2, 반지x2)당:

| 필드 | 입력 타입 | 값 |
|------|----------|-----|
| 티어 | dropdown | T4 유물, T4 고대 |
| 전투 특성 수치 | number input | 부위별 범위 (목걸이: 15178~17857, 귀걸이: 11806~13889, 반지: 10962~12897) |
| 연마 옵션 1~3 | dropdown | 아래 연마 옵션 목록 참조 |

### 연마 옵션 (각 3등급: 하/중/상)
- 추가 피해: +0.70% / +1.60% / +2.60%
- 적에게 주는 피해: +0.55% / +1.20% / +2.00%
- 무기 공격력(%): +0.80% / +1.80% / +3.00%
- 공격력(%): +0.40% / +0.95% / +1.55%
- 치명타 적중률: +0.40% / +0.95% / +1.55%
- 치명타 피해: +1.10% / +2.40% / +4.00%
- 무기 공격력(정수): +195 / +480 / +960
- 공격력(정수): +80 / +195 / +390

---

## 5. 팔찌

| 필드 | 입력 타입 | 값 |
|------|----------|-----|
| 티어 | dropdown | T4 유물, T4 고대 |
| 스탯 종류 1~4 | dropdown | 없음, 힘, 민첩, 지능, 치명, 특화, 신속, 제압, 인내, 숙련, 체력 |
| 스탯 수치 1~4 | number input | 0~ |
| 팔찌 효과 1~4 | dropdown | 없음 + 15개 카테고리별 3등급 효과 |

### 팔찌 효과 카테고리
- 공이속, 치적+치주피, 치피+치주피, 적주피, 적주피+무력화
- 쿨+적주피, 추피+악마, 적중시 무공+공이속, 무공+조건부무공
- 무공+스택무공, 백어택피해, 헤드어택피해, 타대피해, 추가피해, 아군공강

---

## 6. 어빌리티 스톤

|------|----------|-----|
| 스톤 각인 1 | dropdown | 없음 + 각인 목록 x Lv.1~4 |
| 스톤 각인 2 | dropdown | (위와 동일) |
| 세공 보너스 | dropdown | 0, 1175, 2350, 3525 |

## 7. 각인 (5슬롯)

각 슬롯당:

| 필드 | 입력 타입 | 값 |
|------|----------|-----|
| 각인 등급 | dropdown | 유물, 전설 |
| 각인 레벨 | dropdown | X0~X4 |
| 각인 종류 | dropdown | 없음 + 전체 각인 목록 (45개 일반 + 4개 감소 + 직업 각인) |

---

## 8. 아크패시브 (3트리)

각 트리(진화, 깨달음, 도약)당:

| 필드 | 입력 타입 | 값 |
|------|----------|-----|
| 카르마 랭크 | radio | 0~6 |
| 카르마 레벨 | number + stepper | 1~30 |

**일괄 변경:**
- 내실 포인트: 없음(0), 일반1(3), 일반2(6), 일반3(9), 상급(5), 상급+일반1(8), 상급+일반2(11), 상급+일반3(14)
- 랭크 일괄: 0~6
- 레벨 일괄: 1~30

---

## 9. 아크그리드 (6슬롯)

각 슬롯당:

| 필드 | 입력 타입 | 값 |
|------|----------|-----|
| 코어 등급 | dropdown | 영웅, 유물, 고대 |
| 블레이드 종류 | dropdown | 없음 + 그리드별 3~6개 스킬 옵션 (클래스/그리드 종류에 따라 다름) |
| 슬롯 효과 (4개) | 편집/삭제 | 효과 아이콘 표시 |

**일괄 변경:** 코어 등급 일괄, 포인트 일괄 (10P~20P)

---

## LOPEC에 없는 항목

- 카드 세트 변경
- 스킬 트리 변경
- 전투 특성 직접 입력 (악세 스탯으로 간접 조절)

## 우선순위 (효율 분석에 미치는 영향 순)

1. **각인** — 딜증에 직접적 영향, 구현 간단
2. **보석** — 스킬 딜증/쿨감에 영향
3. **악세서리 연마** — 공격력/치적/치피 등 핵심 스탯
4. **아크패시브 랭크/레벨** — 포인트 총량 변경
5. **팔찌** — 복합 효과
6. **장비 재련/품질** — 공격력 기반
7. **아크그리드** — 코어 효과
8. **어빌리티 스톤** — 각인 레벨 보너스
9. **아바타** — 미미한 스탯 보너스
