> 최종 수정: 2026-04-03

## 서비스 구성

| 탭 | 설명 | 상태 |
|----|------|------|
| 능력치 | 장비, 보석, 특성, 각인, 아크패시브, 아크그리드, 카드 | ✅ 완료 |
| 스킬 | 스킬 트리 (딜지분 계산은 추후) | 미구현 |

| **시뮬레이터** | 원본 데이터 주입 → 장비/세팅 변경 → 상태 저장 | 미구현 |
| **효율 분석** | 시뮬레이터 상태 기반 LOPEC 동일 항목 계산 | 미구현 |
